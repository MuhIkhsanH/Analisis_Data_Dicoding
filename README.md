# How to Setup

### Setup Envoirment Anaconda
```
conda create --name bikesharing python=3.9
conda activate bikesharing
pip install -r requirements.txt
```


### Setup Envoirment Terminal
```
mkdir bikesharing
cd bikesharing
pipenv install
pipenv shell
pip install -r requirements.txt
```

### Run Steamlit App Local
```
streamlit run dashboard.py
```
